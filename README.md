
## Strawberry Wallpaper 草莓壁纸

> 采用elctron-vue开发的壁纸应用，适用于mac端

#### 开发
```
npm install
npm run dev
```
#### 打包
1.先执行webpack打包页面
```
npm run build-r
```
2.按平台打包 `electron-builder` 会默认当前平台进行打包
```
npm run build-mac
npm run build-win
```
